\chapter{Αποσπάσματα Κώδικα και \en{openHAB Rules}}

Ακολουθεί ενδεικτικό απόσπασμα κανόνα (\en{Rule}) που υλοποιήθηκε στο περιβάλλον \en{openHAB} για τον έλεγχο της
ακουστικής άνεσης (\en{NoiseGuard}) στην εκάστοτε σχολική αίθουσα. Ο κώδικας καταγράφει τις υπερβάσεις θορύβου και
ενεργοποιεί την οπτική ανατροφοδότηση.

\begin{lstlisting}[caption={Listing Γ.1: Ενδεικτικός κανόνας NoiseGuard σε openHAB},label={lst:noiseguard_rule}]
var int cl1Count = 0

rule "Noise threshold CL1"
when
    Item CL1_Noise_dB changed
then
    val db = (CL1_Noise_dB.state as Number).floatValue
    val th = (CL1_Noise_Threshold.state as Number).floatValue
end
\end{lstlisting}

\begin{lstlisting}[caption={Listing Γ.2: Ενδεικτικός κανόνας FridgeGuardian σε openHAB},label={lst:fridgeguardian_rule}]
rule "Fridge Door Alert"
when
    Item Canteen_Fridge_Door changed to OPEN
then
    createTimer(now.plusSeconds(45), [ |
        if(Canteen_Fridge_Door.state == OPEN) {
            Canteen_Buzzer_Cmd.sendCommand(ON)
            logInfo("Canteen", "Fridge open > 45s!")
        }
    ])
end
\end{lstlisting}
